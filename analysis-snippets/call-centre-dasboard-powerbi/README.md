# üìû Call Centre Performance Dashboard (Power BI)

This repository hosts a professional Business Intelligence project featuring an interactive Power BI dashboard designed for Call Centre management. The dashboard provides a visual, real-time-like view of operational efficiency and service performance.

## üéØ Project Goal

To create a comprehensive, single-page dashboard that allows managers to quickly monitor key performance indicators (KPIs), identify service bottlenecks, and make data-driven decisions to improve call centre operations and customer satisfaction.

## üìà Dashboard Highlights & Metrics

The Power BI file showcases advanced data modeling and visualization techniques, focusing on the following core Call Centre metrics:

| Category | Key Metrics | Insights Provided |
| :--- | :--- | :--- |
| **Service Level** | Service Level %, Abandonment Rate, First Call Resolution (FCR) | Gauging the speed and effectiveness of customer service delivery. |
| **Efficiency** | Average Handle Time (AHT), Average Talk Time, Average Waiting Time | Identifying opportunities to optimize agent performance and process flow. |
| **Volume & Demand** | Total Calls Handled, Call Volume by Time of Day/Week, Top Agents/Locations | Understanding workload distribution and resource allocation needs. |
| **Customer Satisfaction**| Customer Satisfaction (CSAT) Score, Top Resolution Topics | Measuring the quality of service and identifying areas for agent training. |

* **DAX (Data Analysis Expressions):** Utilized for creating complex measures and calculated columns (e.g., Service Level %) to ensure accurate and dynamic KPI calculations.
