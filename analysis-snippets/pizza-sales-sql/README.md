# Pizza Sales SQL Analysis — Mini Project

**Author:** Akshada Pimpariya  
**Tools:** SQL, PostgreSQL / MySQL  

## Overview
This project analyzes pizza store sales using SQL queries.  
It includes analysis of revenue, orders, top pizzas, categories, and window functions.

## Files
- `pizza.sql` — main SQL file
- `dataset` - dataset

## How to run
Open `pizza.sql` in any SQL editor (PgAdmin, MySQL Workbench)  
and run queries one by one.
